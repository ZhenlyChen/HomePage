# HomePage

主页生成器

根据配置文件生成自己的个人主页，展示个人信息。

## Function

- 多种主题、自定义主题(基于 EJS 模板)
- 识别并编译 Less 文件为 CSS
- 多语言支持

## Usage

### Step 1

编辑配置文件 `config.json`

### Step 2

生成静态文件

```bash
npm run build
```

## Themes

### green-mountain

经典主题，翠绿山峦

- 根据 RSS 资源动态渲染博客最新文章
- 展示个人项目
